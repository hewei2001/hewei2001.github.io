本文最后更新于：2024年10月23日 下午

记录一些常用的 Prompt，包括：交互式论文撰写、科研论文的润色、文生图 Prompt 优化等。

本文提供的 Prompt 参考了部分公开文档，并结合了一些自己日常使用的经验。

交互式论文撰写

现有的 LLMs 已经拥有足够强大的多语言能力，以下内容直接用中文进行输入即可，推荐使用 GPT-4o。

从头开始写

注意事项：

  • 开头必须强调英文学术论文，否则模型可能用中文输出
  • 关键点尽量按照你希望呈现的顺序，用中文罗列出来即可，越详细越好

续写已有段落

注意事项：

  • 已有的英文段落可以直接贴上去，让其续写风格更统一

批注式修改

注意事项：

  • 批注式修改需要紧接着上一轮对话使用，上一轮可以是：生成、续写、修改；
  • 直接在上一轮输出的英文句子用中文括号批注即可，例如（这句话可以再简练一些）；
  • 也可以自己修改一些英文句子，甚至直接用中文改上去，再让其优化；
  • 如果上一轮对话已经找不到了怎么办？多加一些 Context 即可。

优化、缩写

注意事项：

  • 建议每段话之后都让其优化一下

实验与分析

注意事项：

  • 实验分析的标题很重要，如果不会想的话可以带上第一个关键点
  • 如果没有实验结果表格，只有图片呢？只需要将图片的 Caption 输入，并且描述图片的大致内容，就能起到类似的效果

专业 Prompt 生成

适合任务相对复杂，而自己经验或专业知识比较少，导致无法准确描述需求的情况；或者你想要 ChatGPT 生成一个更个人化、定制化的回答时。

上面那段话直接发给 ChatGPT 无需修改，然后等它回复之后再给它一个最初级的 Prompt 让他修改，例如：

  • 让 ChatGPT 写一篇关于初学者如何学习 Pytorch 的教程
  • 让 ChatGPT 帮我用 Python 写一个实现计算器程序

再根据它的回复逐步提出新问题，逐步完善，最终得到一个完整提示。
