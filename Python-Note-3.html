全局配置 rcParams

Matplotlib 使用 MRC（Matplotlib Resource Configurations）配置文件来自定义各种属性，我们称之为 rc 配置或者 rc 参数。使用 rcParams 可以控制几乎所有的默认属性：视图窗口、线条、颜色、样式、坐标轴、网格、文本、字体等属性。
